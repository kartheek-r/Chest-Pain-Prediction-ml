# Chest-Pain-Prediction-ml
ML project to predict musculoskeletal chest pain using mental health factors

## 📌 Project Overview
This project predicts musculoskeletal chest pain severity using mental health and lifestyle factors like anxiety, stress, sleep, and activity.

## 🎯 Objective
To analyze how psychological and lifestyle factors affect chest pain and build a predictive model.

## 📊 Dataset
Source: Kaggle  
Dataset includes:
- Anxiety Score
- Depression Score
- Stress Level
- Sleep Hours
- Physical Activity

## ⚙️ Methodology
- Data Preprocessing
- Feature Engineering (Created Pain_Level)
- Model: Random Forest Classifier
- Evaluation: Accuracy, Confusion Matrix

## 📈 Results
- Accuracy: ~65–70%
- Key Insights:
  - Stress and anxiety increase pain
  - Sleep and activity reduce pain
